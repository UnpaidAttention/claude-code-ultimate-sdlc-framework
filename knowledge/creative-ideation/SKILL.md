# Creative Ideation

Structured approaches to generating innovative ideas. Moves beyond conventional thinking to discover novel solutions and opportunities.

## Creative Thinking Principles

### 1. Defer Judgment
Separate idea generation from evaluation. Criticism kills creativity.

### 2. Quantity Over Quality
Generate many ideas first. Quality emerges from quantity.

### 3. Welcome Wild Ideas
Impractical ideas spark practical innovations.

### 4. Build on Others
Combine, extend, and transform ideas.

### 5. Be Visual
Sketch, diagram, prototype. Make ideas tangible.

## SCAMPER Method

Apply these prompts to existing features or problems:

### S - Substitute
- What can be replaced?
- What other materials, processes, approaches?
- What other user interface paradigm?

**Example prompts:**
- What if we substituted voice for typing?
- What if we substituted AI for manual curation?
- What if we substituted async for sync?

### C - Combine
- What can be merged?
- What features can blend?
- What categories can combine?

**Example prompts:**
- What if we combined search and navigation?
- What if we combined analytics with actions?
- What if we combined documentation with the interface?

### A - Adapt
- What can be modified for this purpose?
- What from other domains applies?
- What metaphors from the real world?

**Example prompts:**
- How does [unrelated industry] solve this?
- What if we adapted game mechanics?
- What natural process could we mimic?

### M - Modify/Magnify/Minimize
- What can be changed in form, shape, meaning?
- What can be made bigger, faster, stronger?
- What can be made smaller, simpler, optional?

**Example prompts:**
- What if this was 10x faster?
- What if this was 10x simpler?
- What if we exaggerated this feature?

### P - Put to Other Uses
- What else could this do?
- What other users could benefit?
- What other contexts apply?

**Example prompts:**
- What if power users had this for beginners?
- What if this feature served a different job-to-be-done?
- What if we exposed this internally as an API?

### E - Eliminate
- What can be removed?
- What is unnecessary?
- What would happen without this?

**Example prompts:**
- What if we removed the signup process?
- What if we eliminated settings entirely?
- What's the simplest version that works?

### R - Rearrange/Reverse
- What can be reordered?
- What if we reversed the flow?
- What if roles were swapped?

**Example prompts:**
- What if users did the admin functions?
- What if we started at the end?
- What if the system asked questions first?

## Lateral Thinking Techniques

### Random Entry
1. Select a random word/object/image
2. List its attributes
3. Force connections to the problem
4. Explore unexpected insights

**Example:**
- Random word: "Restaurant"
- Attributes: Menu, reservation, chef, courses, ambiance
- Apply to software: What if our feature had a "menu" of options? What if there was a "reservation" system for resources?

### Reverse Assumptions
1. List all assumptions about the problem
2. Reverse each assumption
3. Explore what the reversal enables

**Example:**
- Assumption: "Users must log in to access content"
- Reversal: "Users can access all content without logging in"
- Explore: What value does the account provide then? How do we convert anonymous to registered?

### Analogy Thinking
1. Define the core problem
2. Find analogous problems in other domains
3. Study how those domains solve it
4. Translate solutions back

**Example:**
- Problem: Users don't complete onboarding
- Analogy: Video games (tutorials, levels, rewards)
- Translate: Progressive disclosure, achievements, rewards

## Brainstorming Protocol

### Phase 1: Diverge (15-30 min)
- Generate as many ideas as possible
- No criticism or evaluation
- Build on others' ideas
- Encourage wild ideas

### Phase 2: Cluster (10 min)
- Group similar ideas
- Identify themes
- Name clusters

### Phase 3: Evaluate (15 min)
- Rate on feasibility (1-5)
- Rate on impact (1-5)
- Plot on 2x2 matrix

### Phase 4: Select (10 min)
- Choose ideas to develop
- Assign ownership
- Define next steps

## Creative Exercises

### "What If" Exploration
Ask "What if..." questions to unlock possibilities:
- What if we had unlimited budget?
- What if we had to ship tomorrow?
- What if we were building for children?
- What if this was a game?
- What if there was no screen?

### First Principles Thinking
1. Break down to fundamental truths
2. Reason up from there
3. Ignore existing solutions

### Constraint Relaxation
1. Identify constraints
2. Remove each one mentally
3. Explore what's possible without it
4. Find ways to relax the constraint
